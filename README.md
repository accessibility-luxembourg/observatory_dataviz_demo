# Digital accessibility Luxembourg data visualization components
This page presents six diagrams created in CSS using semantic and accessible content, coded using only the following HTML tags:

`<table>`
`<dl>`, `<dt>`, `<dd>`
`<ul>`, `<li>`
  
These HTML elements are then “repainted” using CSS.

For more information, please go to this page: https://accessibility-luxembourg.github.io/observatory_dataviz_demo/
